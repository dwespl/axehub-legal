AxeHub - Privacy Policy

Last updated: April 19, 2026

Summary: AxeHub does not collect, transmit, or sell any personal data. All data stays on your device and your local network.

1. Data We Collect

None. AxeHub does not collect any personal information, analytics, telemetry, or usage data. The app has no backend servers, no user accounts, and no cloud services.

2. Local Data Storage

AxeHub stores the following data locally on your device only:

  • Miner IP addresses — to connect to your mining devices on the local network
  • Mining statistics — hashrate history, share counts, best difficulty (for charts)
  • App settings — language, currency, pool type, thresholds
  • Tuning profiles — frequency/voltage configurations you create or import
  • Schedule configurations — day/night mining schedules
  • Hub connection tokens — for remote access pairing (stored locally)
  • Nano3S credentials — username and password hash for device authentication
  • Diagnostic logs — masked (IP addresses, wallet addresses, and MAC addresses are obfuscated before storage)

This data is stored using Hive (local database) and SharedPreferences. It never leaves your device unless you explicitly use the Hub feature to share miner status with another device you control.

3. Network Communication

AxeHub communicates over the network in the following ways:

  • Local network (LAN): HTTP requests to your mining devices (Bitaxe port 80, Nano3S port 4028). This traffic stays within your home network.
  • Hub mode: If you enable Hub mode, the app runs an HTTPS server on your device (port 8742) that other instances of AxeHub can connect to. This uses self-signed TLS certificates with certificate pinning. You control who can connect via 6-digit pairing codes.
  • Price data: The app fetches cryptocurrency prices from CoinGecko API and mining difficulty from WhatToMine. These are public APIs; no personal data is sent — only the coin ID and fiat currency code.

4. Third-Party Services

AxeHub uses the following third-party APIs for price and difficulty data:

  • CoinGecko (privacy policy)
  • WhatToMine (whattomine.com)

The following third-party API is optional and opt-in — contacted only if you enable the weather scheduler feature and provide your own API key:

  • OpenWeatherMap (privacy policy) — when enabled, AxeHub sends the city name or coordinates you enter in Settings, together with your personal API key, to api.openweathermap.org to retrieve current weather conditions. AxeHub does not request device location permission; the location data is whatever you typed manually. Disable the feature to stop all contact with this service.

No personal data, device identifiers, or usage statistics are shared with any of these services.

5. Permissions

  • Internet — to communicate with miners on your LAN and fetch price data
  • Notifications — for temperature alerts, offline warnings, and milestone celebrations
  • Camera — only when you use the QR scanner to import tuning profiles (not used otherwise)
  • Foreground Service — to keep the Hub server running in the background on Android (opt-in)

6. Children's Privacy

AxeHub is not directed at children under 13. The app manages cryptocurrency mining hardware and is intended for adult users.

7. Data Deletion

All data is stored locally. Uninstalling the app removes all stored data. You can also clear individual data (miners, logs, profiles) from within the app settings.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated date.
